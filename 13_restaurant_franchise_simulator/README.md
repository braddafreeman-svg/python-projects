# Restaurant Franchise Simulator

## Overview
A Python project that models a restaurant business using object-oriented programming (OOP).

It includes:
- `Menu` objects with items, hours, and a bill calculator
- `Franchise` locations with available menus by time
- `Business` objects that group franchises together

## Skills Practised
- Classes and objects
- Instance attributes and methods
- `__repr__` for readable object output
- Dictionaries for menu items/prices
- Lists of objects and iteration
- Basic OOP modelling
