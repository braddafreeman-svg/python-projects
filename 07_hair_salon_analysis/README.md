# Hair Salon Analysis

## Overview
A beginner Python project that analyses hair salon pricing and revenue data.

The script calculates:
- average haircut price
- updated discounted prices
- total weekly revenue
- average daily revenue
- list of hairstyles under $30 after discounts

## Skills Practised
- Lists
- Loops (for loops)
- List comprehensions
- Indexing
- Basic calculations and totals
- Printing formatted output

## How to Run
Run the Python file:

```bash
python salon_analysis.py
