# Gradebook Project

This project creates and updates a student gradebook using Python lists.

It stores subjects and grades, updates values, removes entries, and combines gradebooks from multiple semesters into one full record.

## Skills Practised
- Lists
- Nested lists
- Adding and removing list items
- Updating list values
- Printing formatted output

## How to Run

Run the Python file:

```bash
python gradebook.py
