# Pizza Shop Project

## Overview
A beginner Python project that works with lists to manage pizza toppings and prices.  
The program pairs pizza names with their prices, sorts them, and finds the cheapest options.

## Skills Practised
- Lists
- Counting values in lists
- Sorting lists
- Indexing and slicing
- Removing list items (`pop()`)
- Printing output

## How to Run
Run the Python file:

```bash
python pizza.py
