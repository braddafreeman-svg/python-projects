# Password Security Project

## Overview
A Python project that reads compromised usernames from a CSV file and outputs them into a text file.

It also creates a JSON message for the boss and writes a scrambled ASCII signature into a new CSV file.

## Skills Practised
- Reading CSV files using `csv.DictReader`
- Writing text files (`.txt`)
- Writing JSON files using `json.dump()`
- File handling with `with open(...)`
- Working with lists and loops

## How to Run
Run the Python file:

```bash
python password_security.py
