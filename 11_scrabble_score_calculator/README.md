# Scrabble Score Calculator

## Overview
A Python project that calculates Scrabble-style word scores using a letter-to-points dictionary.

The script can:
- score individual words
- calculate total points for multiple players
- store final results in a dictionary

## Skills Practised
- Dictionaries
- `zip()` and dictionary comprehensions
- Functions
- Loops and nested loops
- Using `.items()` to iterate through dictionaries
- Using `.get()` to safely retrieve values

## How to Run
Run the Python file:

```bash
python scrabble_score.py
