# Physics Functions Project

## Overview
A beginner Python project that practises writing and using functions to calculate values related to physics.

It includes:
- temperature conversion (Fahrenheit ↔ Celsius)
- force calculation (F = m × a)
- energy calculation (E = mc²)
- work calculation (Work = Force × Distance)

## Skills Practised
- Writing functions with parameters
- Returning values
- Default parameter values
- Using functions inside other functions
- String formatting with `.format()`

## How to Run
Run the Python file:

```bash
python physics_functions.py
