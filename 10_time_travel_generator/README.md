# Time Travel Generator

## Overview
A Python project that generates a fun time travel scenario using the current date/time, random destinations, and a randomly chosen target year.

The script calculates a fictional travel cost based on the year difference and outputs a formatted time travel message.

## Skills Practised
- Importing modules
- `datetime` for current date/time
- `decimal.Decimal` for accurate currency calculations
- Random generation (`randint`, `choice`)
- Custom modules and functions

## How to Run
Run the Python file:

```bash
python time_travel.py
